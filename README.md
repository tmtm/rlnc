# Rlnc

Netcat with readline.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rlnc'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rlnc

## Usage

```
% rlnc hostname port
```
